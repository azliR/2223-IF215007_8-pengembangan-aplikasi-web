# React Intro

## Instalasi
- Download NodeJS
  - Windows : https://nodejs.org/en/download/
  - Linux
    - Ubuntu : apt-get install nodejs

- Cek Instalasi NodeJS
  ```sh
  node --version
  
  npm --version
  ```
  
- Install Yarn
  ```sh
  node install -g yarn
  ```
  
- Buat project React
  ```sh
  yarn create react-app nama-aplikasinya
  ```
